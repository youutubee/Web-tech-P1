Website Design Specification for Woox Travel Clone
General Overview
A travel/tour agency multipage website.

Clean, professional, and modern layout.

Fully responsive with Bootstrap 5.2.0 framework.

Key features: homepage carousel slider, reservation form, tour listings, contact page with map.

Layout Structure
Header

Logo left aligned.

Navigation menu on the right with smooth scrolling or page links.

Sticky navigation on scroll.

Homepage

Full-width carousel slider with large hero images.

Overlay text and call-to-action (CTA) buttons on sliders.

Section for popular tour destinations with image cards.

Reservation form section with fields: Name, Email, Phone, Destination, Date, etc.

About section with text and an image.

Footer with social media icons, quick links, and copyright.

Other Pages

Tours listing page with grid or list layouts showing multiple tours.

Tour detail page with large images, description, itinerary, price, and booking button.

Contact page with a contact form and integrated Google Map.

Design Palette (Colors)
Primary Color: #00bcd4 (Cyan/Teal used on buttons, highlights, and links)

Secondary Color: #007b8a (Darker shade for hover states and secondary elements)

Background Color: #ffffff (White for most backgrounds)

Text Color: #333333 (Dark gray for readability)

Accent Color: #f5f5f5 (Light gray for backgrounds behind sections/cards)

Button Color Normal: #00bcd4

Button Color Hover: #007b8a

Typography
Font Family: "Open Sans", sans-serif (loaded via Google Fonts)

Headings: Bold, varying sizes from H1 large (48px) to H6 small (14px)

Body Text: Regular, 16px size for readability

Button Text: Uppercase, bold, white color on colored backgrounds

Buttons
Primary Buttons

Background: #00bcd4

Text: White, uppercase, bold

Padding: 12px 30px

Border Radius: 4px

Hover: Background #007b8a, smooth transition 0.3s

Secondary Buttons (if any)

Transparent background with colored border (#00bcd4)

Text color #00bcd4

Hover: Filled with #00bcd4 and white text

Forms
Input Fields

Border: 1px solid #ccc

Border Radius: 4px

Padding: 10px

Focus: Border color changes to #00bcd4

Submit Button

Uses primary button style

Images & Media
Carousel slider images: Full width, cover style, height around 70vh

Tour cards images: Uniform size, rounded corners (4px radius)

Responsive scaling for different devices

Icons & Extras
Use FontAwesome or similar icon library for social media and UI icons

Smooth scroll animations for navigation

Hover effects on cards: Slight shadow and scale-up

Responsive Design
Navigation collapses into hamburger menu on small screens

Layout stacks vertically on mobile devices

Touch-friendly buttons and forms
