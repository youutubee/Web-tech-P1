# Images Directory

This directory is for storing image assets used in the VoyageFlow project.

## Image Sources

All images are currently loaded from Unsplash CDN. For production, you should:

1. Download the images
2. Optimize them (compress, resize)
3. Store them in this directory
4. Update the image paths in the HTML/CSS files

## Image Requirements

- **Carousel Images**: 1920x1080px minimum, landscape orientation
- **Destination Cards**: 800x600px, landscape orientation
- **Tour Cards**: 400x300px, landscape orientation
- **Formats**: JPG or WebP for photos, PNG for icons
